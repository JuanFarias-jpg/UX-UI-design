Acerca de

  • ¿Quienes somos?

    Este sitio web es un proyecto desarrollado por estudiantes como parte de la materia Interfaz Web. Su objetivo es ofrecer una plataforma informativa sobre los mundiales de fútbol, permitiendo consultar datos, estadísticas y curiosidades de cada edición. Además, cuenta con un espacio de publicaciones para usuarios, donde se pueden compartir opiniones, análisis y comentarios relacionados con los torneos.
    Este proyecto combina diseño, funcionalidad y experiencia de usuario, aplicando los conocimientos adquiridos durante el curso.

Términos de uso

  • Leer más

    Al acceder o utilizar este sitio, el usuario acepta los siguientes términos:

    Uso académico: La página tiene fines informativos y educativos. No representa una entidad comercial ni oficial.

    La información sobre los mundiales y las publicaciones de los usuarios se presenta con propósitos ilustrativos y no debe considerarse fuente oficial.

    Propiedad intelectual: El diseño y el contenido del sitio fueron creados únicamente para evaluación académica.

    Limitación de responsabilidad: Los desarrolladores no se hacen responsables por el mal uso de la información ni por daños derivados del uso del sitio.

    Modificaciones: Dado que se trata de un proyecto en desarrollo, el contenido, diseño y funcionamiento pueden cambiar sin previo aviso.


    El uso de esta página implica la aceptación de estos términos y condiciones.


Privacidad

  • Leer más

    Este sitio web ha sido desarrollado por estudiantes como parte del proyecto académico de la materia Interfaz Web.

    El propósito de esta página es mostrar información sobre los mundiales de fútbol y permitir a los usuarios realizar publicaciones relacionadas con el tema.

    No se recopilan datos personales sensibles ni se realiza ningún tipo de tratamiento con fines comerciales.

    En caso de que el usuario proporcione información (como nombre o correo electrónico en formularios o comentarios), estos datos no se guardan en ninguna base de datos ya que este es solo un prototipo.

    Este proyecto no almacena información privada de manera permanente ni comparte datos con terceros.

    El uso de este sitio implica la aceptación de este Aviso de Privacidad con fines exclusivamente educativos.
